# Financial-Risk-Compliance-Analysis-of-Big-4-Firms
**Executive Summary**

This project presents a structured analysis of financial risk and compliance performance across four major global audit firms: Deloitte, EY, PwC, and KPMG. The dashboard provides a consolidated view of key risk indicators, enabling stakeholders to assess operational efficiency, compliance exposure, and audit effectiveness.

**Objective**

The objective of this analysis is to:

Evaluate fraud detection and violation trends
Measure revenue impact due to compliance gaps
Assess audit effectiveness across firms
Identify high-risk industries
Analyze the role of AI in modern auditing practices
Methodology

**The analysis was conducted using a structured data pipeline:**

Data cleaning and transformation using Power Query
Data modeling to establish relationships across key metrics
Development of calculated measures using DAX
Visualization through an interactive Power BI dashboard
Key Findings
Firms with higher compliance violations demonstrate increased revenue loss exposure
Audit effectiveness varies significantly across organizations, indicating process maturity differences
Healthcare and Technology sectors show elevated risk concentration
AI adoption in auditing correlates with improved fraud detection capability
Dashboard Components

**The report includes:**

KPI indicators for performance monitoring
Comparative analysis across firms
Trend analysis for revenue loss and audit effectiveness
Industry-level risk segmentation
AI utilization and compliance distribution metrics
Tools & Technologies
Power BI
DAX (Data Analysis Expressions)
Power Query
Data Modeling Techniques
Data Source

The dataset used in this project is simulated and designed to reflect realistic financial risk and compliance scenarios.
